為高中生網頁程式營隊所寫的教材

2019/07

這個課程介紹了基本的網頁程式設計，並結合 Webduino Bit 發展板製作了一個小遊戲。

本課程包括以下內容：

  • 基本的 HTML 文件結構
  • 基本的 CSS 語法規則
  • 基本的 JavaScript 語法規則
  • 基本的 canvas 繪圖程式設計 (使用 Konva)
  • 基本的 Webduino Bit 程式設計
  • 整合 Webduino Bit 的網頁程式設計
