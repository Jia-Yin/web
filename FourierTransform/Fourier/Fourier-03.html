# 傅立葉轉換-03

在上一個小單元中，你是否發現當頻率成份為 1 1 1 1 時，時間取樣為 4 0 0 0 呢？其實我們只要看第二個取樣時間點的結果就可以知道了。下面的圖應該會在第二個取樣時間點暫停。現在我們來觀察左邊的圖，直流的成份波還是停在 1 的位置；頻率 1 的成份波跑了 1/4 圈；頻率 2 的成份波跑了 2/4 圈；頻率 3 的成份波跑了 3/4 圈。因為這 4 個點剛好是對稱的，它們的和為 0，當然實數部份也是 0，所以取樣值就是 0 了。
