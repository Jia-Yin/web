# 傅立葉轉換實驗-05

現在我們來看一點數學。考慮 N 個跑者，有 N 個取樣時間點的問題。如果全部的振幅都是 1，開始的角度都是 0，那麼取樣的結果第一個值應該是 N，其他 N 個值都會是 0。

那麼怎樣讓第 k 個取樣值是 N，而其他值是 0 呢？先看 k=1 的情況，也就是第一個取樣時間點才重合，那我們可以把速度 1 的倒退 1/N 圈，速度 2 的倒退 2/N 圈，依此類推就可以了。換句話說，他們的起始位置應該是：

1,e−i2π/N,e−i4π/N,⋯,e−i2(N−1)π/N1, e^{-i2\pi/N}, e^{-i4\pi/N}, \cdots, e^{-i2(N-1)\pi/N} 1,e−i2π/N,e−i4π/N,⋯,e−i2(N−1)π/N

這邊 eiθe^{i\theta}eiθ 表示在單位圓上，角度為 θ\thetaθ 的位置（尤拉公式：eiθ=cos⁡(θ)+isin⁡(θ)e^{i\theta} = \cos(\theta)+i\sin(\theta)eiθ=cos(θ)+isin(θ)），請參照下圖（引自 betterexplained.com）。 e−i2π/Ne^{-i2\pi/N}e−i2π/N 就是倒退 1/N 圈的位置。
