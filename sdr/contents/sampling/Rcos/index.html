# 升餘弦脈衝信號

以下以升餘弦脈衝信號為例，來進行抽樣定理的簡單分析。假設 s(t)s(t)s(t) 為 α=1\alpha=1α=1 的升餘弦脈衝信號，

s(t)=sinc⁡(t/T)cos⁡(πt/T)1−4t2/T2=sin⁡(2πt/T)(2πt/T)(1−4t2/T2)s(t)=\frac{\operatorname{sinc}(t/T)\cos(\pi t/T)}{1-4t^2/T^2} = \frac{\sin(2\pi t/T)}{(2\pi t/T)(1-4t^2/T^2)} s(t)=1−4t2/T2sinc(t/T)cos(πt/T)​=(2πt/T)(1−4t2/T2)sin(2πt/T)​

其傅立葉變換經計算後為

S(f)={Tcos⁡2(πfT/2),∣f∣<1/T0,∣f∣>1/T S(f) = \begin{cases} T\cos^2(\pi fT/2),& |f| < 1/T\\ 0,& |f| > 1/T \end{cases} S(f)={Tcos2(πfT/2),0,​∣f∣<1/T∣f∣>1/T​

取 T=2T=2T=2，則

s(t)=sin⁡(πt)/(πt(1−t2))s(t)=\sin(\pi t)/(\pi t(1-t^2)) s(t)=sin(πt)/(πt(1−t2))

S(f)={2cos⁡2(πf),∣f∣<1/20,∣f∣>1/2 S(f) = \begin{cases} 2\cos^2(\pi f),& |f| < 1/2\\ 0,& |f| > 1/2 \end{cases} S(f)={2cos2(πf),0,​∣f∣<1/2∣f∣>1/2​

以下為了方便觀察，我們來撰寫一個計算時域的函數，取名為 mys.m，程式碼如下：

function y = mys(x)
% Calculate raise cos with a=1 and T=2

pos1 = x==-1 | x==1; % |x|=1 須特別處理 
pos0 = x==0; % x=0 須特別處理
x(pos0) = 99;
x(pos1) = 99;
y = sin(pi*x)./(pi*x.*(1-x.*x));
y(pos0) = 1;
y(pos1) = 0.5;
