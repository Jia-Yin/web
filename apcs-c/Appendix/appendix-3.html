# 附錄三：APCS 檢測環境安裝

這邊要介紹的是APCS檢測環境的安裝，讀者在操作之前，應先下載以下檔案：

  1. Oracle VM VirtualBox：https://www.virtualbox.org/wiki/Downloads
  2. APCSPractice.iso：https://apcs.csie.ntnu.edu.tw/index.php/info/environment/ 最下面的連結

下載完成之後，按以下步驟進行安裝：

  1. 安裝Oracle VM VirtualBox：

    以下這個畫面是關於USB的部份，建議安裝

  2. 安裝完成後，按新增，以便建立虛擬機器。

  3. 名稱建議填寫APCS，類型選擇Linux，版本選Ubuntu (64-bit)：

  4. 記憶體的部份，如果寬裕的話，可以選擇2048MB，如果比較緊的話就用1024MB。

  5. 不要加入虛擬硬碟。我們會直接用ISO檔模擬光碟，並使用光碟機開機執行檢測環境。

    不要理會這個警告：

  6. 安裝完成之後按「設定」：

  7. 選擇「存放裝置」，把滑鼠移到「控制器：IDE」後面兩個圖案上，看提示文字，選擇新增光碟：

  8. 出現以下畫面，按「選擇磁碟」：

  9. 接著按「加入」，選擇已下載的APCSPractice.iso檔案：

  10. 完成之後回到VirtualBox主畫面，在APCS的虛擬機器上按啟動，開機完成之後，就會進入以下畫面，接著就可以打開Code::Blocks開始進行程式碼的編寫了。

  11. 要結束虛擬機的時候，可以點選螢幕最右下方的按鈕，會出現以下畫面，接著點選「Shutdown」即可。
