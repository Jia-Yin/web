# 傅立叶转换

非周期性函数的频率成份是连续的，我们可使用傅立叶转换来计算其频率密度函数，假设时域函数为 s(t)s(t)s(t)，其傅立叶转换公式如下：

S(f)=∫−∞∞f(x)e−j2πftdtS(f) = \int_{-\infty}^{\infty}f(x)\ e^{-j2\pi ft}\ dtS(f)=∫−∞∞​f(x) e−j2πft dt

一般在信号处理时，我们多使用快速傅立叶转换 (FFT) 来计算其频谱，使用快速傅立叶转换时，应牢记以下几个重点：

  1. 假设用来分析的信号段有 NNN 个点，且其取样间隔为 Δt\Delta tΔt，则此信号段总长为 T=NΔtT=N\Delta tT=NΔt，且取样速度为 R=1/ΔtR=1/\Delta tR=1/Δt。

  2. 根据取样定理，可以观察的频率范围为 [−f0,f0][-f_0,f_0][−f0​,f0​]，f0=R/2f_0=R/2f0​=R/2。

  3. NNN 个时域点做 FFT 之后变成 NNN 个频域点，故频率解析度，亦即相邻频率点的间隔为 2f0/N=R/N=1/(NΔt)=1/T2f_0/N = R/N = 1/(N\Delta t) = 1/T2f0​/N=R/N=1/(NΔt)=1/T。
    简单说，取样时间总长的倒数就是频率取样点之间的距离，也就是频率解析度。

  4. 要记得取样速度的一半，是可观察的最大频率；取样总长的倒数是频率解析度。

  5. 如果是实函数的话，做完 FFT 之后，其正频率和负频率相对的值会是共轭复数。
