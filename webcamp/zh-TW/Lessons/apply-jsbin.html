# 申請 jsbin 帳號

  1. 開啟 https://jsbin.com
  2. 在右上角點選 Login or Register

  1. 如果你已經有 Github 帳號，可以直接登入，否則點選 Create an account

  1. 填寫註冊資訊，接著點選 Verify account

  1. 旋轉圖形直到朝上為止，然後按完成

  1. 再次點選 Create ac account

  1. 選擇 Free 的帳號方案

  1. 接下來要使用 Email 確認帳號

  1. 登入 Github 帳號，授權給 jsbin 使用

  1. 檢查帳號資訊是否正確


練習

  • 你可以閱讀 JS Bin features 學習更多 jsbin 的使用方式 (閱讀 Getting started 以及 Keyboard shortcuts.)
  • 在本課程中，我們會使用 jsbin 以及 jsFiddle 做為開發學習的平台。這邊先介紹 jsbin 的原因，主要是因為 jsbin 可以讓我們發佈設計好的頁面。
