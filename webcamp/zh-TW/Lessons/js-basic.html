# JavaScript

JavaScript (JS) 是可以在瀏覽器執行的程式語言，你可以用它來改變頁面的內容和樣式，也可以用來控制頁面的操作邏輯。

# 如何插入 JS 代碼

# 放在 HTML 中

你可以在 HTML 檔案中把 JS 代碼放在 的標記中間。例如：

The answer is: <span id="demo"></span>

<script>
document.getElementById("demo").innerHTML = 5 + 6;
</script>

# 放置在外部檔案

如果你使用 jsbin 或 jsFiddle 之類的線上平台，平台會自動插入並執行 JavaScript 中的代碼。如果你自己在本地端撰寫檔案，則使用以下語法來插入和執行你的JS 代碼檔案。(這裡假設你的 JS 檔案名稱為 myscripts.js)

<script src="myscripts.js"></script>
