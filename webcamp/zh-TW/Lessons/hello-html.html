# 基本的 HTML 文件結構

  • <!DOCTYPE html> : 說明這是 HTML5 文件
  • <html>...</html> : 文件首尾的標記符號，裡面為文件內容
  • <head>...</head> : 標記內為所有用來輔助頁面內容的 非可見元素
  • <body>...</body> : 標記內為頁面所有的 可見的結構元素

通常在 head 標記內會看到以下內容

  • <meta charset="UTF-8"/> : 表示此檔案使用 UTF-8 的字元編碼
  • <title>...</title> : 定義本頁面的視窗標題
# 範例
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8" />
  <title>--TITLE--</title>
</head>
<body>
  Hello!
</body>
</html>
