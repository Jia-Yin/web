# About The Course

# Overview

In this course, you will learn the following topics:

  • Basic HTML document structure
  • Basic CSS syntax
  • Basic JavaScript syntax
  • Basic canvas-drawing programming (with Konva)
  • Basic Webduino bit programming
  • An integrated project with Web programming and Webduino Bit

We will introduce some basic knowledges for web design. Then we will introduce the Webduino Bit controller. At last, we will combine all together to make an integrated project.
