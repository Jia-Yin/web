# 3. IFTTT 的 Webhook 服務

Webhook 指的是網頁相關的外掛，也就是透過網頁介面所提供的服務。在 IFTTT 中，Webhook 可以用來做為事件觸發，也可以當做事件發生後所需的回應。

這一小節主要參考 Buber 的 Webduino使用LINE自訂積木(一)：IFTTT，基本上是利用 Webduino 呼叫 Webhook，做為觸發 IFTTT 的事件，另外使用 LINE Notify 做為事件的回應，也就是 Webhook 觸發之後，會把相關的資訊透過 LINE 發送出來。

練習

  1. Buber 的文稿中，除了把訊息傳到 LINE 之外，也提到把訊息傳到 Messenger 的作法，有 Messenger 帳號的，可以試看看。

  4. Buber 的第二篇相關文稿 Webduino使用LINE自訂積木(二)：LINE Notify，是利用 LINE Notify 本身提供的服務，以及 Buber 所寫的 LINE Notify 積木，直接把訊息從 Webduino 送到 LINE Notify，沒有透過 IFTTT，有興趣的讀者可以試看看。
