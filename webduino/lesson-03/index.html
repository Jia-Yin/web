# Webduino 與物聯網應用 - Line自訂積木

在這個單元，我們要介紹 Webduino 和物聯網平台連結的相關應用，我們將會以Buber筆記中的 「Webduino使用Line自訂積木」做為學習主軸。在本單元中，將會學到：

  1. IFTTT 網站服務
  2. Webduino 和 IFTTT 連結的相關應用
  3. 用 Line notify 來發送訊息
  4. 使用 Line Bot 來控制 Webduino 開發板

在Buber筆記中的 「Webduino使用Line自訂積木」中，共有四篇文稿：

  1. Webduino使用LINE自訂積木(一)：IFTTT
  2. Webduino使用LINE自訂積木(二)：LINE Notify
  3. Webduino使用LINE自訂積木(三)：LINE Bot
  4. Webduino使用LINE自訂積木(四)：LINE Bot

這一個單元只會介紹其中的 (一) 和 (三)，想進一步學習的讀者可直接參考Buber筆記中其他相關的連結，甚至也可以學習其他不是 Webduino 相關的文稿。

另外，為了遵重智慧財產權，也因為 Buber 筆記文稿對於操作步驟描述得非常清楚，因此在學習第 (一) 和 (三) 篇文稿時，除了一些基本的介紹之外，會直接連進 Buber 的筆記中，依其步驟進行操作。
