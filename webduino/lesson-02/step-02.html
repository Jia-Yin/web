# 2 哪裡有擴充模組

  • Webduino 有一個 技術討論社團
  • 裡面有一位 法蘭斯 的成員
  • 他寫了很多 Webduino 的 擴充模組
  • 如果懂得一些相關技術，你也可以自己寫擴充模組

法蘭斯的網站中，有一個說明檔，詳細說明在 這裡，我們一起研究一下。

  • 台灣空氣品質：可取得政府開放資料中的空氣品質資料
  • Bit 全彩點矩陣：針對 Bit 開發板（Bit 板內建週邊比較多，但 Smart 板擴充接腳比較多）
  • 顯示文字新增：設定文字大小、樣式、顏色等
  • 溫濕度變化圖(D3.js)：可用來繪製曲線圖
  • 遊戲元素：可以用來製作一些簡單的遊戲（Scratch 還是比較好用，以後如果有 Scratch 的 Webduino 擴充元件會比較理想）
  • Bit：針對 Bit 開發板整理的擴充模組，有一些和其他的分類應該是重覆的
  • 顯示網頁：使用 iframe 內嵌顯示其他網頁（必須是 https 開頭的網頁，有的還不給嵌呢！）
  • IFTTT：If this then that，一個新生的網絡服務平台，通過其他不同平台的條件來決定是否執行下一條命令
  • 模擬指示燈：模擬點矩陣燈的元件（Bit 有一個實際的）
  • Smart Uart MyFirmata：UART 指的是序列埠的通訊，初學者可以略過
  • 朗讀語言擴充：Google 語音發聲或辨識的擴充套件
  • 機械學習(Google深度學習)：深度學習的一個嘗試模組（以影像為例）
  • thingspeak：ThingSpeak 是一個 IoT 的伺服器服務，這個模組應該是用來讀寫資料
  • Line Bot：用來傳送訊息到 Line 的擴充模組，如果你不是程式開發人員，可以跳過了
  • Google Script：與 Line 搭配的 Google Scripts，如果你不是程式開發人員，可以跳過
  • 裝置狀態 (斷線後可自動連線與重新執行主程式)：偵測裝置狀態，斷線也可以自動偵測和重連
  • Javascript 指令擴充：擴充了幾個 JavaScript 指令的模組
  • 監看程式碼(開啟瀏覽器主控台觀看程式碼)：用主控台 (console) 觀看程式用的，但似乎不能使用？
  • ESP8266：無線模組的通訊模組，不是程式開發人員，可以跳過
  • WebBit：似乎是透過 Web 介面控制 Bit 的燈號顯示的模組
