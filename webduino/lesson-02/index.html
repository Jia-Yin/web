# Webduino 擴充模組介紹與應用

Webduino 的開發平台，已經提供了為數不少的程式積木可以使用，但實際上，我們也可以設計自己的積本，來擴充它的功能；或者，我們也可以直接使用別人設計好的擴充模組。

在這個單元中，我們將會學習到以下的一些內容：

  1. 學習 Webduino 擴充模組的匯入與使用。
  2. 學習專案管理與分享。
  3. 學習使用溫溼度繪圖擴充模組與朗讀文字擴充模組。
  4. 學習流程的使用方法。

另外，我們在最後一個小單元，利用擴充模組寫了一個小小的應用，你也可以連到以下的網頁玩看看。（必須先連上 Smart 板，然後開啟網頁後填入 Device ID）
https://output.jsbin.com/zedotuz
